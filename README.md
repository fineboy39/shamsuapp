# Health Fellow Chatting App

## Project Description
The Health Fellow Chatting App is designed to facilitate communication among health fellows and professionals. It aims to provide a platform for sharing information, discussing cases, and collaboration on health-related topics.

## Features
- **User Authentication:** Secure login and registration for users.
- **Chat Functionality:** Real-time messaging with user presence indicators.
- **Group Chats:** Allow users to create and join group discussions.
- **File Sharing:** Users can share documents and images relevant to their discussions.

## Technologies Used
- **Frontend:** React.js
- **Backend:** Node.js with Express
- **Database:** MongoDB
- **Real-time Communication:** Socket.io

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/fineboy39/shamsuapp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd shamsuapp
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage
- Users can create an account and log in to access the chat features.
- Join or create groups to start discussing with peers.
